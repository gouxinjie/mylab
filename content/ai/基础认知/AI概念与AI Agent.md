# AI 概念与 AI Agent

## 什么是 AI

人工智能（Artificial Intelligence，AI）是让机器具备"感知、理解、推理、决策"能力的技术总称。当下最主流的一支是以**大语言模型（LLM）**为核心的生成式 AI，它通过海量文本训练，学会预测下一个最可能出现的词，从而完成对话、写作、翻译、编程等任务。

## AI Agent 是什么

**AI Agent（智能体）**是在 LLM 之上构建的、能够"自主完成任务"的系统。相比只会一问一答的聊天机器人，Agent 具备以下能力：

- **规划（Planning）**：把复杂目标拆解为一系列可执行的子步骤
- **工具调用（Tool Use）**：调用搜索、代码执行、数据库等外部工具
- **记忆（Memory）**：记住上下文与历史交互，支持长期任务
- **反思（Reflection）**：根据执行结果自我纠错、迭代优化

## 一个简化的 Agent 循环

```text
观察(Observation) → 思考(Thought) → 行动(Action) → 观察结果 → ...循环直至完成目标
```

## Chatbot 与 Agent 的区别

| 维度 | 传统 Chatbot | AI Agent |
| --- | --- | --- |
| 交互方式 | 一问一答 | 自主多步执行 |
| 是否用工具 | 否 | 是 |
| 是否有记忆 | 弱 | 强 |
| 适用场景 | 问答、闲聊 | 自动化任务、复杂工作流 |

> 一句话总结：**LLM 是大脑，Agent 是给大脑装上了手、脚和记忆。**
