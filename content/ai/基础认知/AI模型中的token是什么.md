# AI 模型中的 token 是什么

## 定义

**Token（词元）**是大语言模型处理文本的最小单位。模型并不是按"字"或"词"来理解文本，而是先把文本切分成一个个 token，再转换成向量进行计算。

## token 的粒度

一个 token 可能是：

- 一个完整的英文单词，如 `apple`
- 一个单词的一部分，如 `un` + `believable`
- 一个汉字或词的一部分
- 一个标点符号或空格

经验参考：

- 英文中，**1 个 token ≈ 4 个字符 ≈ 0.75 个单词**
- 中文中，**1 个汉字 ≈ 1~2 个 token**

## 为什么 token 很重要

1. **上下文窗口（Context Window）**：模型一次能处理的 token 数量有上限，例如 128K token。超出部分会被截断或遗忘。
2. **计费单位**：绝大多数 API 按 token 计费，区分输入 token 与输出 token。
3. **性能与延迟**：token 越多，计算量越大，响应越慢。

## 估算示例

```text
输入："你好，帮我写一段代码"  →  约 10 个 token
一篇 1000 字的中文文章       →  约 1500~2000 个 token
```

> 优化提示词、精简上下文，本质上就是在**节省 token**，既省钱又提速。
